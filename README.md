# KC Frontrunners Routes

Canonical route data and GPX assets for the KC Frontrunners app.

## Structure

- `data/route-curation.csv`: human-edited route source of truth
  - includes route metadata such as display copy, Garmin links, and `origin` (`loose_park`, `mill_creek`, `sunday`)
- `data/routes.json`: app-facing generated route feed
- `gpx/`: hosted GPX route files
